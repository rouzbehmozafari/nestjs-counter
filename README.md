## Description

Simple counter Server.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

````bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

## Run tests

```bash
# unit tests
$ pnpm run test
````
